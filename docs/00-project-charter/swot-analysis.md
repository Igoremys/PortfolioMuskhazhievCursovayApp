# SWOT-анализ

Короткий SWOT-анализ проекта YPhoto.

## Strengths (Сильные стороны)
- Полнота документации (00–12).
- Мобильное приложение с оффлайн-режимом (Room).
- JWT-аутентификация и покрытие тестами backend (>40% в JaCoCo).

## Weaknesses (Слабые стороны)
- Отсутствуют формальные артефакты планирования (WBS / Gantt / COCOMO).
- Нет матрицы стейкхолдеров в документации (исправлено добавлением).
- Ограниченная автоматизация статического анализа (частично отсутствует отчет).

## Opportunities (Возможности)
- Добавление CI-отчётов (SonarQube/Checkstyle) и метрик в README улучшит восприятие.
- Внедрение WBS/Gantt/COCOMO повысит итоговый процент проверки.

## Threats (Угрозы)
- Риск несоответствия нормативам безопасности при публикации фото.
- Возможные проблемы с синхронизацией данных в оффлайн-сценариях.
