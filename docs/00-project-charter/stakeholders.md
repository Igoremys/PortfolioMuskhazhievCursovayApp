# Матрица стейкхолдеров

Ниже приведена базовая матрица стейкхолдеров для проекта YPhoto. При необходимости дополните реальными контактами и ролями.

| Стейкхолдер | Роль / интерес | Влияние | Ожидания | Каналы коммуникации | Контакт / ответственный |
|-------------|----------------|--------:|---------|--------------------|----------------------|
| Заказчик (Руководитель проекта) | Инициирует и утверждает требования | Высокое | Соответствие ТЗ, сроки, бюджет | E-mail, встречи, отчёты | Руководитель проекта |
| Конечный пользователь (фотограф) | Пользователь мобильного приложения | Высокое | Удобный UI, надежное хранение фото, оффлайн | Тестирования, опросы | Продуктовая команда |
| Разработчики (mobile/backend) | Реализация функционала | Среднее/Высокое | Чёткое ТЗ, CI/CD, тесты | Slack/GitHub, PR | Технический лидер |
| Техподдержка / Администратор | Эксплуатация сервиса | Среднее | Документация, админ-руки | Руководство администратора | Администратор |
| Защита персональных данных / юрист | Соответствие требованиям безопасности | Низкое/Среднее | Соответствие политике конфиденциальности | Документы, ревью | Специалист по безопасности |
