# Этап 4: Детальное проектирование

## Общая информация
На данном этапе производится детальное проектирование системы: описывается взаимодействие компонентов через диаграммы последовательности, проектируется детальная структура классов и специфицируются ключевые методы.

## Контекст проектирования
Проект реализован по **Траектории В (Мобильная разработка)** с использованием паттерна **PCMEF**, адаптированного под Android Jetpack:

- **Presentation**: Jetpack Compose экраны
- **Control**: ViewModel с StateFlow
- **Mediator**: Repository (Single Source of Truth)
- **Entity**: Доменные модели (Photo, User)
- **Foundation**: Room (локальный кэш) + Retrofit (REST API)

## Результаты этапа
1.  Диаграммы последовательности (4 сценария)
2.  Диаграмма классов проектирования
3.  Спецификация ключевых методов
4.  Описание примененных паттернов проектирования
