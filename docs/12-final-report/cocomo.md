# Оценка трудоёмкости по COCOMO

Здесь приведена оценка по базовой модели COCOMO для проекта YPhoto на основе фактического объёма кода.

Проект содержит примерно 14.8 KLOC (14790 строк кода). Для модели Basic (Organic) используются коэффициенты:

- a = 2.4
- b = 1.05

Формула:

E = a * (KLOC)^b  (человеко-месяцев)

Расчёт:

E = 2.4 * (14.8)^1.05 ≈ 41 чел.-мес.

График расписания:

TDEV = c * (E)^d, где c = 2.5, d = 0.38

TDEV ≈ 2.5 * 41^0.38 ≈ 10 месяцев.

Вывод:

По базовой модели COCOMO проект YPhoto оценивается в 41 человеко-месяц с плановой продолжительностью около 10 месяцев. Это значение актуально для небольших команд и служит ориентиром при планировании ресурсов.
