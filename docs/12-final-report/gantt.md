# Диаграмма Ганта

Проект YPhoto разбит на ключевые этапы и временные рамки в формате Gantt. Даты отражают плановую последовательность работ.

```mermaid
gantt
    dateFormat  YYYY-MM-DD
    title       План проекта YPhoto
    section Инициация
    Анализ и согласование :a1, 2026-03-01, 2026-03-14
    section Проектирование
    Архитектура и спецификации :a2, 2026-03-15, 2026-03-28
    section Реализация
    Backend development :a3, 2026-03-29, 2026-04-18
    Mobile development  :a4, 2026-03-29, 2026-04-18
    section Тестирование
    Unit/Integration :a5, 2026-04-19, 2026-04-30
    section Развёртывание
    CI/CD и документация :a6, 2026-05-01, 2026-05-10
```

Данный Gantt используется как окончательный план этапов разработки и будет включён в итоговую документацию.
