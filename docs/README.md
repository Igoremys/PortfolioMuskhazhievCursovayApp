# Документация проекта YPhoto

В этом каталоге собраны все артефакты проектирования, требования, реализация и тесты для курсового проекта "YPhoto".

## Структура документации
- `00-project-charter/` — инициация проекта, бизнес-контекст, цели, риски.
- `01-requirements/` — системные требования, Use Case, Domain Model, матрица трассировки.
- `02-architecture/` — архитектура PCMEF, интерфейсы, ADR.
- `03-database/` — логическая модель данных, DDL, ORM-мэппинг.
- `04-design/` — детальное проектирование, диаграммы последовательности и классов.
- `05-implementation/` — структура кода и реализация ядра системы.
- `06-testing/` — тест-планы, JaCoCo, Postman, результаты тестирования.
- `07-refactoring/` — рефакторинг, code smells, улучшения.
- `08-ui/` — UI-спецификация, экраны, скриншоты.
- `09-api/` — документация REST API, OpenAPI/Swagger, описание эндпоинтов.
- `10-deployment/` — среда, CI/CD, развертывание.
- `11-user-guide/` — руководство пользователя.
- `12-final-report/` — финальный отчет, техническая спецификация, руководство администратора.

## Как использовать
Откройте каталог нужной темы и изучите README для базового описания, затем перейдите к специализированным документам по ссылкам.
