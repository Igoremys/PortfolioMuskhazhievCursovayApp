# UI-спецификация

## Экран `HomeScreen`
- Показывает список фотографий из кэша Room.
- Отображает состояние: `Loading`, `Success`, `Empty`, `Error`.
- Кнопка обновления запускает синхронизацию с сервером.

## Экран `ProfileScreen`
- Показывает данные текущего пользователя: имя, email, био, аватар.
- Список собственных фотографий.
- Кнопка `Edit Profile` открывает экран редактирования.

## Экран `AddPhotoScreen`
- Поле для выбора изображения.
- Поля `Title`, `Description`.
- Кнопка `Publish` отправляет фото на сервер или сохраняет локально при отсутствии сети.

## Взаимодействие с карточкой фото (`ModernPhotoCard`)
- При нажатии на иконку лайка происходит оптимистическое обновление.
- При нажатии на фото открывается подробный просмотр или экран комментариев.

## Навигация
- Bottom navigation или drawer не используются, основной доступ реализован через экраны `Home`, `Profile`, `Add`.
- Переходы между экранами осуществляются через Navigation Component / Compose Navigation.
