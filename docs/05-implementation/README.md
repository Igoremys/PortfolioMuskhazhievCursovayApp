# Этап 5: Реализация ядра системы

## Общая информация
На данном этапе производится физическая реализация спроектированной архитектуры. Ядро системы написано с соблюдением принципов SOLID и паттерна PCMEF. 

**Технологический стек реализации:**
- **Сервер (Backend):** Java 17, Spring Boot 3, Spring Data JPA, PostgreSQL, Spring Security (JWT).
- **Клиент (Android):** Kotlin, Jetpack Compose, Room, Retrofit, Coroutines & Flow.
- **Тестирование:** JUnit 5, JaCoCo (для отчёта о покрытии).

## Результаты этапа
1.  Исходный код ядра (слои Entity, Foundation, Mediator)
2.  Модульные тесты (Unit Tests) для бизнес-логики
3.  Отчёт о покрытии кода (JaCoCo Report, покрытие > 40%)
4.  Документация по структуре проекта и тестированию
