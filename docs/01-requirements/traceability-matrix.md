# Матрица трассировки требований

Матрица связывает Бизнес-прецеденты (BUC из Этапа 0) с Системными прецедентами (UC из Этапа 1).

| ID BUC (Бизнес-требование) | Название BUC | ID UC (Системное требование) | Название UC | Приоритет |
|----------------------------|--------------|------------------------------|-------------|-----------|
| BUC 1 | Аутентификация и регистрация | UC1 | Аутентификация и регистрация (JWT) | Высокий |
| BUC 2 | Создание и редактирование портфолио | UC3 | Загрузка и публикация фото | Высокий |
| BUC 2 | Создание и редактирование портфолио | UC4 | Управление своими фото (CRUD) | Высокий |
| BUC 3 | Публикация медиа-контента | UC3 | Загрузка и публикация фото | Высокий |
| BUC 4 | Поиск и просмотр работ | UC2 | Просмотр ленты фотографий | Высокий |
| BUC 4 | Поиск и просмотр работ | UC7 | Поиск фото и пользователей | Средний |
| BUC 5 | Социальное взаимодействие | UC5 | Постановка лайка | Средний |
| BUC 6 | Оффлайн-режим (кэширование) | UC6 | Оффлайн-кэширование и синхронизация | Критический |
| BUC 7 | Модерация контента | UC8 | Модерация контента | Низкий |

## Статус реализации UC'ов

| UC ID | Статус | Примечание |
|-------|--------|-----------|
| UC1 | ✅ Полностью реализовано | JWT аутентификация работает |
| UC2 | ✅ Полностью реализовано | Лента фото из кэша с синхронизацией |
| UC3 | ✅ Полностью реализовано | Загрузка работает, но без очереди при оффлайн |
| UC4 | ⚠️ Частично реализовано | Delete реализовано, Edit API есть но нет UI |
| UC5 | ✅ Полностью реализовано | Лайки с оптимистическим обновлением |
| UC6 | ⚠️ Упрощенная реализация | Кэш работает, но нет WorkManager/BroadcastReceiver для фоновой синхронизации |
| UC7 | ✅ Полностью реализовано | Поиск фото и пользователей работает |
| UC8 | ❌ НЕ реализовано | Модерация контента отсутствует |
