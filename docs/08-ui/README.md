# UI-документация

В этом разделе описывается пользовательский интерфейс мобильного приложения YPhoto. Подробности включают спецификацию экранов, навигацию и скриншоты.

## Содержание
- `screenshots.md` — визуальное отображение экранов и ключевых взаимодействий.
- `ui-spec.md` — техническая спецификация интерфейсов и экранов.
