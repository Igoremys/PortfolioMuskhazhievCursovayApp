# Руководство администратора

## Основные функции администратора
- модерирование контента;
- управление пользователями;
- проверка статусных данных и логов сервера.

## Модерация
- Администратор может просматривать все фото и помечать нежелательный контент.
- Отмеченные фото не отображаются в общей ленте до проверки.

## Управление пользователями
- Администратор может просматривать список аккаунтов.
- Проверять подозрительные регистрации.
- Блокировать пользователей при необходимости.

## Поддержка системы
- Контроль состояния сервера и базы данных.
- Мониторинг логов и ошибок.
- Обеспечение резервного копирования данных.
