# Финальный отчет проекта YPhoto

## Цель проекта
Разработка мобильного приложения для управления фотопортфолио с поддержкой оффлайн-режима и синхронизации данных.

## Ключевые результаты
- Создано Android-приложение на Kotlin с использованием Jetpack Compose.
- Реализован сервер на Java Spring Boot с REST API и JWT-аутентификацией.
- Добавлен локальный кэш через Room и стратегия оффлайн-синхронизации.
- Разработан паттерн PCMEF для разделения слоев приложения.
- Подготовлена документация по требованиям, архитектуре, дизайну и тестированию.

## Достижения
- Поддержка публикации, редактирования и удаления фото.
- Оптимистичное обновление лайков.
- Работа приложения при отсутствии сети.
- Автоматические тесты и отчет JaCoCo.
