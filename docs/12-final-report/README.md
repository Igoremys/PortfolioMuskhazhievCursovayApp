# Финальная документация

Раздел содержит итоговый отчет по проекту YPhoto, техническую спецификацию, руководство администратора и пользовательское руководство.

## Содержание
- `admin-guide.md` — руководство администратора.
- `final-report.md` — итоговое описание проекта.
- `technical-specification.md` — техническая спецификация решения.
- `user-guide.md` — пользовательское руководство.
