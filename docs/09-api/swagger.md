# Swagger UI

Swagger UI позволяет интерактивно просматривать и тестировать REST API сервера.

## Запуск
1. Запустите сервер Spring Boot локально.
2. Откройте браузер.
3. Перейдите на URL: `http://localhost:8080/swagger-ui/index.html`

## Использование
- Выберите нужный эндпоинт.
- Укажите параметры запроса и тело.
- Добавьте заголовок `Authorization` при необходимости.
- Нажмите `Execute`.

## Преимущества
- Наглядная документация всех доступных API.
- Возможность отправлять запросы без написания кода.
- Легкое тестирование сценариев регистрации, входа, публикации фото и лайков.
