# Руководство пользователя

## Быстрый старт
1. Запустите приложение на Android-устройстве или эмуляторе.
2. Зарегистрируйтесь или войдите в уже существующий аккаунт.
3. После входа вы попадете на главный экран с лентой фотографий.

## Основные действия

### Просмотр ленты
- На главном экране отображается список последних фото.
- Фото берутся из кэша и обновляются из сети.
- При отсутствии сети приложение продолжает работать на локальных данных.

### Публикация нового фото
1. Нажмите кнопку `+` для добавления фото.
2. Выберите изображение из галереи или камеры.
3. Заполните `Title` и `Description`.
4. Нажмите `Publish`.

### Лайк фотографии
- Нажмите на иконку лайка на карточке фото.
- Счетчик лайков обновится мгновенно благодаря оптимистичному обновлению.

### Редактирование профиля
1. Откройте экран профиля.
2. Нажмите `Edit`.
3. Измените имя, описание или аватар.
4. Сохраните изменения.

### Оффлайн-режим
- Если интернет отключен, приложение продолжает показывать данные из Room.
- Публикации и лайки сохраняются локально и синхронизируются после восстановления связи.
