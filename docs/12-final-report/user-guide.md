# Пользовательское руководство (финальное)

## Введение
Приложение YPhoto предназначено для фотографов и авторов контента, которые хотят управлять портфолио, публиковать фото и получать обратную связь.

## Как начать
1. Зарегистрируйтесь или войдите через существующую учетную запись.
2. Перейдите на главный экран, чтобы просмотреть последние публикации.
3. Для создания нового фото нажмите кнопку `+`.

## Основные возможности
- публикация фотографий;
- просмотр ленты и поиск контента;
- лайки и взаимодействие с фото;
- редактирование профиля;
- оффлайн-доступ и синхронизация.

## Рекомендации
- Используйте стабильное соединение для загрузки больших изображений.
- В оффлайн-режиме фото сохраняются локально и отправляются на сервер после восстановления сети.
- Для корректной работы авторизации следите за актуальностью JWT-токена.
