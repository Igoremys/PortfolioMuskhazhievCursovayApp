# API-документация

В этом разделе описаны REST API сервера проекта YPhoto, включая список эндпоинтов, спецификацию OpenAPI и инструкции по использованию Swagger UI.

## Содержание
- `endpoints.md` — описание основных HTTP-эндпоинтов.
- `openapi.md` — принципы OpenAPI и структура API.
- `swagger.md` — использование Swagger UI для тестирования API.
